# Hong-Wei Zhang academic homepage

This is a minimal static academic homepage for GitHub Pages.

## Files

- `index.html`: page content
- `style.css`: visual style
- `.nojekyll`: disables Jekyll processing on GitHub Pages

## Deploy on GitHub Pages

1. Create a repository named `zhangh-w.github.io`, or use any repository with GitHub Pages enabled.
2. Upload these files to the root of the repository.
3. In GitHub, go to **Settings → Pages**.
4. Set source to **Deploy from a branch**, branch `main`, folder `/root`.
5. Visit `https://zhangh-w.github.io/` after GitHub Pages finishes deployment.

## Edit

Edit `index.html` directly. The lines most likely to need updates are:

- position and affiliation near the top;
- institutional email and office address in the contact section;
- publication status and journal references.

This version uses no Jekyll theme, no Ruby, no external JavaScript, and no third-party CSS framework.
